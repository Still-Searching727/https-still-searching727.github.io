Post Mortem

If I had an additional 8 weeks, I would focus more on making a functional puzzle system. All I was able to get done is puzzle piece creation and user generated puzzle files. Using the extensible components, it would be cool to add more functionality to the pieces to make games like tetris or puzzles with more mechanics.

Overall, even though I wish I could have done more, I'm happy to have an engine that uses a scripting language to create various blocks and puzzle pieces.
